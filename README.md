# SmartSavingsHub

Социальный финансовый помощник для коллективного и индивидуального накопления на крупные цели.

## О проекте

**Проблема:** людям тяжело копить на крупные покупки.

Социальный финансовый помощник будет позволять копить как индивидуально,
так и в группах.

## Кому подойдет
- Индивидуальному пользователю или группе людей

## Требования
### FR
#### 1. Управление целями
- Пользователь создает цель, указывая название, сумму, срок и тип (личная/групповая)
- Владелец группы может приглашать других пользователей по email/логину. Участники могут принимать/отклонять приглашения
- Роли: Админ (может редактировать цель, удалять участников) и Участник (может только вносить средства и видеть прогресс)
#### 2. Деньги и расчеты
- Пользователь фиксирует пополнение (симуляцию перевода)
- Система автоматически рассчитывает, сколько внес каждый участник и какую долю от цели он уже закрыл
#### 3. Отображение и мотивация 
- Дашборд прогресса: Общий и Индивидуальный
- Лента событий: Пуш-уведомления
- Напоминания

### NFR
1. Доступность: 99.99%
2. Консистеность данных для групповых целей
3. Безопасность: TLS 1.3, шифрование PII. Проверка прав доступа.

### Out of Scope
1. Реальные деньги: спин-офф. Имитация транзакций
2. Инвестиции: сервис не предполагает инвестировать накопления в акции или облигации.
